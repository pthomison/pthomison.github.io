# pthomison.github.com

[Patrick Thomison's](https://www.linkedin.com/in/patrickthomison) Github Page

Deployed Page: [https://pthomison.github.io](https://pthomison.github.io)

Build Machine Dependencies:
- node
- npm

Build Dependencies:
- [Task](https://taskfile.dev)
- [Webpack](https://webpack.js.org/)
- [Vue.js](https://vuejs.org/)
- [Vue Router](https://router.vuejs.org/)
- [Bootstrap](https://getbootstrap.com/)
- [FontAwesome](https://fontawesome.com/)
- [Sass](https://sass-lang.com/)

Build:
```
task clean build
```

Build And Serve :
```
task clean build serve-build
```

Dev Server:
```
task clean serve
```

Full Clean:
```
task clean-all
```
or
```
# clear builds
task clean

# clear dependencies
task clean-deps
```
